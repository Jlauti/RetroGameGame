# Control Plane Map

Operational artifacts are source-of-truth in `agents/`.

## Core Paths

- `agents/backlog/`: ticket definitions
- `agents/delegations/`: worker task briefs
- `agents/reports/`: worker completion reports
- `agents/qa/`: QA signoff artifacts
- `agents/merge/`: merge manifests
- `agents/status/`: gate history, release boards, readiness snapshots
- `agents/loops/`: loop scope and completion contracts
- `agents/templates/`: canonical templates
- `agents/scripts/`: loop automation and validation scripts

## Explanatory Docs

- `docs/agentic/`: stable process docs

## Runtime/Game Code

- `src/`, `assets/`, `specs/`
