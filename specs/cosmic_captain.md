# Cosmic Captain (Inspired by Captain Comic, 1988)

## Aesthetic Goal
- **Era**: 1980s (DOS Age)
- **Palette**: EGA 16-color
- **Visuals**: Large sprites, multi-screen exploration.
- **Audio**: Melodic square waves.

## Modernized Mechanics
- **Physics**: Modern platforming physics (coyote time, jump buffering).
- **Smooth Transitions**: Fast screen-to-screen loading or seamless scrolling where appropriate.
- **UI**: Modern HUD elements that mimic old styles but provide clearer information.

## Core Loop
- **Side-scrolling exploration** across multiple themed zones.
- **Progression**: Start unarmed, find Blastola Cola to shoot.
- **Collectibles**: Keys, high-jump boots, lantern, wave beam.
- **Health**: Shield-based (12 points, 2 per hit).
- **Goal**: Recover three stolen artifacts.

## World Structure
- 8 themed areas, each with 3 zones.
- Zone transitions act as checkpoints.
