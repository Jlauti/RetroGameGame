# Scripts

Orchestration and validation scripts for the agent control plane.

Key scripts:
- `validate_ticket.py` — validate ticket structure
- `validate_loop.py` — validate loop contract
- `check_wip.py` — check WIP limits
- `sync_agent_workspaces.py` — regenerate agent workspaces (deprecated, see brief.md)
- `gate_queue.py` — run merge gate queue

**Run with `py`** (not `python3`) on Windows.
