## Objective

Implement the first runtime/procgen pass for Nebula's procedural neon ground and boundary direction so the battlefield reads as authored terrain play rather than generic height noise.

Required outcomes:
- procgen/runtime must emit explicit ground motifs instead of relying only on generalized heightfield variation
- runtime must support breakable hazard families, placement zones, and destructibility roles defined by the approved contract
- mixed-density cadence must be visible in generated combat space through combat pockets, relief lanes, and lane-pressure segments
- side cage containment must remain visually and mechanically separate from the interesting combat surfaces that live just inside it
- implementation must preserve direct-fire readability while keeping ricochet as a bonus layer
