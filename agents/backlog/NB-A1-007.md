## Objective

Define the gameplay contract for Nebula's next combat pass so the terrain and projectile systems support a stronger loop instead of acting as passive scenery plus crash-pillars.

Required outcomes:
- the ship should read as following the ground and terrain flow, not floating independently of it
- walls, boundaries, and blocker structures should feel tangible and purposeful in combat
- hex extrusions should add challenge and create useful ricochet surfaces for projectiles
- projectile behavior should become more readable and tactically useful alongside terrain interaction

Locked defaults for this ticket:
- ground follow is `visual skim`
- only explicit `hard blockers` are run-ending crash surfaces
- ricochet is a `skillful bonus`, not the main combat solution
