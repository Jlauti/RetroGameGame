## Objective

Implement the first runtime pass for enemy combat AI and hostile return fire so Nebula encounters become two-sided without losing readability.

Required outcomes:
- enemies move with readable intent instead of acting like passive targets
- enemies can shoot back in a way that creates pressure but remains dodgeable and learnable
- enemy combat integrates cleanly with the current terrain-follow, boundary, and ricochet combat space
- runtime behavior stays stable under multiple active enemies and hostile projectiles
